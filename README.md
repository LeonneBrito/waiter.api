# :computer: How to run

```bash
# Clone Repository
$ git clone https://github.com/LeonneBrito/waiter.api.git
```

```bash
# Install Dependencies
$ yarn

# Run Aplication
$ yarn dev
```

Go to http://localhost:3001/ to see the result.
